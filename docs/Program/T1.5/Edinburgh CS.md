    Pros:

    * 选课自由，每个方向都能选，Arch, OS, ML, NLP, CV等
    * 80学分research-based（其中10写research review，10写research proposal，60为thesis）是入门research很好的开始。

    Cons:

    * UK MSc通病，学制太短
    * 冬天下午就天黑，容易emo
